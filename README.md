
## Dépendances Material UI (MUI)
Ce projet utilise **Material UI (MUI)** pour les composants d’interface.  
Les fichiers MUI **ne sont pas inclus dans le projet** (ils ne sont pas dans le dépôt Git),  
vous devez donc installer les dépendances pour que le projet fonctionne.

npm install @mui/material @emotion/react @emotion/styled

## Composants MUI
- import Card from '@mui/material/Card';
- import CardContent from '@mui/material/CardContent';
- import Button from '@mui/material/Button';
- import Box from '@mui/material/Box';
- import Container from '@mui/material/Container';
- import Typography from '@mui/material/Typography';
- import ToggleButton from '@mui/material/ToggleButton';

## Icones MUI 
- import CheckIcon from '@mui/icons-material/Check';
- import DeleteOutlinedIcon from '@mui/icons-material/DeleteOutlined';

# Démarrer le projet
Assurez-vous d’avoir **Node.js** et **npm** installés.

## Installation des dépendances
npm install

## Lancer le projet
npm run dev
